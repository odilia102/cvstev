DwiZ-Commerce

Keywords search for App Store (dwiz-commerce, dwiz), Google Play (dwiz_commerce)

Tech-Stack

  • Front End : Flutter (2.x), Riverpod (next version of provider, one of the best for state management), Hive (local storage), Clean Architecture, Advanced Navigation (like React Native)
  • Back End : microservices GraphQL(AWS, Lambda, SQS, S3, MongoDB, Redis), Typescript, communication between services using Event Driven Architecture (SQS)

Features

  • OTP, image upload (can use remove background for admin / root), Push Notification, Send Email, roles (user, admin, root), Voucher, caching request, inventory stock, search products, infinite scroll / pagination, sort products
